# Mediapipe to MANO Mesh

From mediapipe hand joint, via inverse kinematics, create MANO hand mesh. Visualize the mesh with Open3D.

## Installation

```
python3.8 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```

## Usage

```
python main.py
```
